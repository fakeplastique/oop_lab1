# Python Sheets

Лабораторна робота #1 з ООП.

## Виконав Бондар Ігор, група К-25

## Опис (варіант 6)

Застосунок реалізує функціонал електронних таблиць з можливістю:
- Введення, збереження та обробки таблиць
- Редагування кількості рядків та стовпчиків
- Обчислення виразів з цілими числами довільної довжини
- Підтримки операцій: +, -, *, /, ^, =, <, >, not, and, or
- Посилань на інші клітинки 
- Синтаксичної перевірки виразів з локалізацією помилок
- Двох режимів відображення: ВИРАЗ / ЗНАЧЕННЯ
- Синхронізація з Google Drive

## Встановлення

1. Перевірте версію Python (3.13+):
```bash
python --version
```

2. Встановіть залежності:
```bash
pip install -r requirements.txt
```

3. Додай `credentials.json` для інтеграції з Google Cloud API.

## Запуск

```bash
python main.py
```
